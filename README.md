#Descripción del Trabajo
El proyecto consiste en el desarrollo de un sitio web personal como portafolio para mostrar mi progreso académico en el primer cuatrimestre de mi grado en Ingeniería Informática. El sitio contiene varias páginas: inicio, sobre mí, tema sobre informática, grado, fundamentos de ingeniería informática, páginas de compañeros y contacto.

El desarrollo se realizó con HTML5 y CSS3, asegurando que el sitio fuera responsivo y fácil de navegar.

Un problema en el desarrollo ha sido el control del diseño responsivo ya que inicialmente el sitio no se adaptaba correctamente a pantallas pequeñas, pero se resolvió con CSS Flexbox y Media Queries.
Otro pequeño problema que he tenido es el control de los enlaces entre páginas que se solucionaron revisando la estructura de carpetas.

Este proyecto me ha servido para refrescar conceptos básicos en el desarrollo Web.
